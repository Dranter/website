Calculator

Advanced calculator with all the needed functions you could need (sin, cos, sqrt, pi...). Has the posibility to change to hex, oct, bin, dec with the last one as default. Built full in Python and with the GUI from the library CustomKinter with a several color interfaces. I learnt how to manage a project, build a calculator for my own use and start my journey as a developer.

AI Fish is an experimental virtual assistant designed to explore how language models can express adaptive personalities.
